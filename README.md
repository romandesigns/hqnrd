# Hotel Quinto Nivel RD

Hotel booking website where guests can book and manage their reservations. The project consists of a public site and a dashboard.
Registered users can access their dashboard in order to track and manage reservations. Hotel admin users can access the dashboard and perform
administrative tasks.

## Features

- Internationalization
- Payments
- Rooms
- User accounts [guests, maintanance, housekeeping, manager, developer, owner]

## Public Site

- Home Page
- Rooms Page
- Room Details Page
- Sign In Page
- Sign Up Page

## Dashboard

- Home Page
- Users Page
  - Add
  - Update
- Reservations Page
  - Add
  - Update
- Reporting Page
  - Add
  - Update
- Payments Page
